A complete directory of all Python technique files in this library. Each file covers one specific concept from W3Schools with examples, key concepts, and links to the relevant days in [TWO_WEEK_PYTHON_HOMEWORK_PLAN.md](TWO_WEEK_PYTHON_HOMEWORK_PLAN.md).

Use this index alongside [W3SCHOOLS_PYTHON_REFERENCES.md](W3SCHOOLS_PYTHON_REFERENCES.md) and [TWO_WEEK_PYTHON_HOMEWORK_PLAN.md](TWO_WEEK_PYTHON_HOMEWORK_PLAN.md).

This index is now ordered first by homework day so you can study in plan order. Technique files that apply to multiple days are listed on each relevant day.
