# Gamertag Python Starter Template (Basic)

This is a minimal starting template for the 14-day porting homework.

## Goal
Build a Python console app that matches the C# project behavior:
- Load gamertags from a file
- Display all gamertags
- Filter tags ending with a number
- Filter tags not starting with a letter or number
- Add a gamertag
- Ask whether to run again

## Structure
- `src/main.py` - app entry point and run-again loop
- `src/gamertag_manager.py` - basic class with core methods
- `data/gamertags.txt` - starter data file

## Run
From this folder:

```powershell
python src/main.py
```

## Day Mapping
- Day 8: Project skeleton and file layout
- Day 9: Data loading and welcome sequence
- Day 10: Display all gamertags
- Day 11: Filter ending with number
- Day 12: Filter not starting with letter/number
- Day 13: Add new gamertag and run-again loop
- Day 14: Error handling and final checks

This template stays intentionally basic so you can improve it day by day.
